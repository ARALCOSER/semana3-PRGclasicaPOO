# 📌 Comparación entre Programación Tradicional y POO
### ✅ *AUTOR:*

Ramiro Alcoser A.

### ✅ *DESCRIPCION:*

En este repositorio consta dos soluciones para un mismo ejemplo (Sistema de gestión de mascotas) utilizando diferentes paradigmas de programación:

- Programación Tradicional.
- Programación Orientada a Objetos.

Ambos programas permiten gestionar información básica del Sistema de gestión de mascotas.

### ✅ *ESTRUCTURA DEL PROYECTO:*

```text
programacion_tradicional/
└── tradicional.py

programacion_poo/
├── libro.py
└── main.py
```

### ✅ *REFLEXION:*

- La programación tradicional se centra en las tareas y procedimientos.

- La Programación Orientada a Objetos POO,  se centra en objetos que representan elementos del mundo real.

En el esarrollo de los programas fue posible observar que la POO facilita la organización del código aunque un poco dificil de comprender y permite representar entidades del mundo real de forma más estructurada.
